WSJ Transcript Tool

这个网页小工具专门处理 transcript。当前版本先支持一件事：把你指定的人名在 Name: 这种段首格式里改成 Markdown 粗体，比如 **Imani Mosise:**。也支持把独立占一行的文本提升成 ## 二级标题。整个处理都在浏览器本地完成，不需要后端。
